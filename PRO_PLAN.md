# ADL ShareFlow — תכנית Pro

> מסמך זה מגדיר את תכונות ה-Pro Plan העתידיות.  
> הגרסה הנוכחית (Free) מאפשרת שימוש בסיסי עם מגבלות; תכנית Pro תסיר מגבלות ותוסיף יכולות מתקדמות.

## ✨ תכונות Pro — עתידיות

### 1. קבוצות ללא הגבלת זמן
- קבוצות אירוע פעילות ללא תאריך פקיעה
- קבוצות שוטפות ללא חיוב חוזר חודשי
- מתאים לקבוצות משפחה / שותפים קבועים

### 2. משתתפים ללא הגבלה
- הסרת תקרת המשתתפים בכל סוגי הקבוצות
- אין Tier upgrades נדרשים

### 3. דוחות וייצוא נתונים
- ייצוא הוצאות ל-CSV / Excel
- גרף הוצאות חודשי לפי קטגוריה
- סיכום שנתי לכל הקבוצות
- ייצוא PDF של סיכום קבוצה

### 4. OCR מתקדם
- סריקה אוטומטית של קבלות מרובות בבת אחת
- זיהוי קטגוריה אוטומטי מהקבלה
- שמירת קבלות מקוריות כתמונות

### 5. התראות חכמות
- תזכורות מותאמות אישית לכל חבר קבוצה
- סיכום שבועי/חודשי אוטומטי
- התראה כשיתרה גדולה מסף מוגדר

### 6. מסכי ניתוח מתקדמים
- Dashboard עם גרפים ומגמות
- השוואה בין קבוצות
- ניתוח תשלומים לאורך זמן

### 7. ניהול משתמשים מתקדם
- הזמנת משתתפים חיצוניים (ללא חשבון)
- תפקידים מותאמים אישית (מנהל, גזבר, צופה)
- ביקורת פעולות (Audit log)

### 8. אינטגרציות
- חיבור ישיר ל-Bit / PayBox API (תשלום בתוך האפליקציה)
- Apple Pay / Google Pay
- ייצוא ל-Google Sheets אוטומטי

## 🔧 דרישות טכניות לשלב 1 (Subscription)

**Backend:**
- טבלת `subscriptions` (user_id, plan, start_date, end_date, status)
- Webhook מ-Stripe לעדכון סטטוס
- Middleware שבודק `user.is_pro` בכל endpoint מוגן

**Flutter:**
- מסך "Upgrade to Pro" עם תיאור תכונות ומחיר
- `in_app_purchase` package לתשלום דרך Apple/Google
- עדכון `AuthState.isPro` מה-JWT

**Apple:**
- הגדרת In-App Purchase ב-App Store Connect
- הגדרת Subscription Group + Price Tiers
